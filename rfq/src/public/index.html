RFQ Service API

This service provides endpoints for managing Request for Quotes (RFQs) and Quotes in the B2B E-commerce platform.

RFQ Endpoints

GET /api/rfqs
Get all RFQs with pagination and filtering options
GET /api/rfqs/:id
Get a specific RFQ by ID
POST /api/rfqs
Create a new RFQ
PUT /api/rfqs/:id
Update an existing RFQ
DELETE /api/rfqs/:id
Delete an RFQ
POST /api/rfqs/:id/vendors
Invite vendors to an RFQ
PUT /api/rfqs/:id/vendors/:vendorId
Update vendor status in an RFQ
GET /api/rfqs/:id/quotes
Get all quotes for an RFQ

Quote Endpoints

GET /api/quotes
Get all quotes with pagination and filtering options
GET /api/quotes/:id
Get a specific quote by ID
POST /api/quotes
Create a new quote
PUT /api/quotes/:id
Update an existing quote
DELETE /api/quotes/:id
Delete a quote

Health Check

GET /health
Check the health status of the RFQ service
